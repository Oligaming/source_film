Install on Android

  1. Push this repo to GitHub — the included workflow deploys the app to GitHub Pages automatically:
    https://<user>.github.io/source_film/ (any static HTTPS host works too).
  2. On the phone, open that URL in Chrome.
  3. Menu ⋮ → Install app (or Add to Home screen).
  4. Launch it from the home screen — it runs fullscreen, works offline, and stores its data on the phone.

Updates are automatic: every push stamps a new version, and the installed app shows an Update button on the next launch. (On another host, bump VERSION in sw.js yourself.)
